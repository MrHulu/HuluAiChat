## Tagline (60 chars max)

## Description (260 chars max)

HuluChat is a desktop AI chat app that puts privacy first. Chat with GPT-4, Claude, DeepSeek, or local models via Ollama - all your data stays on your device. Features include RAG document chat, plugin system, and quick panel for instant access. No telemetry, no tracking, ever.

---

## Features List (for Gallery)

1. 🔒 **Privacy First** - No telemetry, no tracking, all data local
2. 🤖 **Multi-Model** - OpenAI, Claude, DeepSeek, Ollama in one app
3. 📄 **Document Chat** - RAG for PDFs and files
4. ⚡ **Quick Panel** - Global hotkey for instant access
5. 🎨 **Beautiful UI** - Dark/light themes, modern design
6. 💾 **Session Management** - Folders, tags, search
7. 📤 **Export** - Markdown export for all conversations
8. ⌨️ **Keyboard Shortcuts** - Built for power users
9. 🧩 **Plugin System** - Extend with custom features
10. 🌐 **Cross-Platform** - Windows, macOS, Linux
