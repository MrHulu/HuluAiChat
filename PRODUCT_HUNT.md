### Description (Extended - for Maker Comment)
HuluChat is a desktop AI chat application designed for developers, writers, and productivity enthusiasts who want a fast, native experience.

**Key Features:**
- 🤖 Multi-model support - Switch between GPT-4, Claude, DeepSeek, Gemini instantly
- 🧠 RAG Smart Q&A - Upload documents and chat with your knowledge base
- 🔌 Plugin System - Extend functionality with official or custom plugins
- 📁 Conversation folders - Organize your chats like files
- ⌨️ Keyboard-first - Do everything without touching your mouse
- 🔍 Search everything - Find any message across all conversations
- 📤 Export anywhere - MD, JSON, or TXT formats
- 🌙 Dark mode - Easy on the eyes, day or night
- 🌐 18 languages - English, 中文, 日本語, 한국어, and more
- ⚡ Native performance - Built with Tauri 2.0, runs at native speed
- 🔄 Auto-updates - Always have the latest features

**Why HuluChat?**
I built HuluChat because I was tired of browser-based AI chat tools that felt slow and cluttered. I wanted something that felt like a native app should - fast, keyboard-accessible, and beautifully designed.

With the latest updates, HuluChat now supports RAG (chat with your documents) and a plugin system for extending functionality. All while staying open source and free.

**Tech Stack:**
- Tauri 2.0 (Rust + WebView)
- React 19 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- FastAPI (Python) backend
