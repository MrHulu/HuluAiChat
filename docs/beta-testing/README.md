# HuluChat Beta Testing Program

> **Version**: v3.73.0
> **Date**: 2026-03-17

## Overview

HuluChat is a privacy-first AI chat desktop application with multi-model support, RAG capabilities, and plugin system.

### Key Features

- **Multi-Model Support**: OpenAI, DeepSeek, GLM, Local Models (Ollama)
- **RAG Knowledge Base**: Chat with your documents
- **QuickPanel**: Global shortcut for instant AI access
- **Privacy First**: No telemetry, no analytics, all data stays local
- **Cross-Platform**: Windows, macOS, Linux
