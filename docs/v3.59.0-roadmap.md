# v3.59.0 Roadmap - 全局快捷唤起 + 技术韧性

> 版本主题：效率优先 + 稳定性增强
> 创建日期：2026-03-12
> 预计周期：5-6 Cycles
> 状态：✅ 已通过 Critic 审核

---

## 版本概要

| 维度 | 内容 |
|------|------|
| **主题** | 全局快捷唤起 + 技术韧性 |
| **核心价值** | 随时唤起 AI，无需切换窗口 |
| **差异化** | Alice AI 的核心卖点，桌面原生实现 |
| **隐私契合** | 完全本地化，无服务器 |

---

## 决策过程

### 原始选项

| 方案 | 差异化 | 风险 | Critic 评分 |
|------|--------|------|-------------|
| 多模型实时对比 | 极强 | 🔴 高 | 需修改 |
| 协作与分享 | 强 | 🟡 中 | 需修改 |
| **全局快捷唤起** | 强 | 🟢 低 | **通过** |

### Critic 逆向审核结论

> **多模型实时对比**：API 成本翻倍是硬伤，建议延后或改为"历史对比"
> **全局快捷唤起**：风险最低，价值明确，可以直接开始

### 最终决策

采纳 Critic 建议，v3.59.0 聚焦：
1. **全局快捷唤起** - 主要功能（P0）
2. **技术债务清理** - 并行进行（P1）
3. **多模型历史对比** - P2 延后到 v3.60.0

---

## MVP 功能范围

### Phase 1: 全局快捷唤起 (P0) - 4 Cycles

#### TASK-211: 全局热键注册
**描述**: 系统级快捷键，应用在后台也能响应

**技术方案**:
- 使用 Tauri global-shortcut 插件
- 默认快捷键: `Ctrl+Shift+Space` (Windows/Linux), `Cmd+Shift+Space` (macOS)
- 支持自定义快捷键
- 冲突检测

**验收标准**:
- [ ] 应用最小化时快捷键生效
- [ ] 与系统快捷键无冲突
- [ ] 可在设置中自定义
- [ ] 冲突时显示警告

**隐私约束**: 仅本地存储快捷键配置

**预计周期**: 1 Cycle

---

#### TASK-212: 快速提问面板
**描述**: 浮动小窗口，快速输入问题

**功能**:
- 小型输入框（不占据全屏）
- 快速发送按钮
- 回车发送，Esc 关闭
- 显示最近回复
- 响应式高度

**UI 设计**:
- 宽度: 400px
- 高度: 自适应（最大 300px）
- 圆角、阴影、毛玻璃效果
- 暗色/亮色主题适配

**验收标准**:
- [ ] 快捷键唤起面板
- [ ] 输入框自动聚焦
- [ ] 发送后显示加载状态
- [ ] 回复显示在面板中
- [ ] Esc 关闭面板

**预计周期**: 1 Cycle

---

#### TASK-213: 剪贴板增强
**描述**: 选中文本 → AI 处理 → 结果回剪贴板

**功能**:
- 检测剪贴板内容
- 选中文本后自动填充到输入框
- AI 回复可一键复制

**Quick Actions 预设**:
| Action | 描述 | 快捷键 |
|--------|------|--------|
| 翻译 | 翻译选中文字 | Ctrl+1 |
| 摘要 | 总结关键要点 | Ctrl+2 |
| 润色 | 改善文字表达 | Ctrl+3 |
| 解释 | 解释概念/代码 | Ctrl+4 |
| 代码审查 | 检查代码问题 | Ctrl+5 |

**验收标准**:
- [ ] 复制文本后自动填充
- [ ] Quick Actions 一键处理
- [ ] 结果可复制到剪贴板
- [ ] 自定义 Quick Actions

**预计周期**: 1 Cycle

---

#### TASK-214: 权限引导
**描述**: 首次启动时引导用户授权

**平台差异**:
| 平台 | 所需权限 | 处理方式 |
|------|----------|----------|
| macOS | 辅助功能权限 | 引导弹窗 + 设置跳转 |
| Windows | 无需额外权限 | 直接使用 |
| Linux | 无需额外权限 | 直接使用 |

**验收标准**:
- [ ] macOS 检测辅助功能权限
- [ ] 未授权时显示引导弹窗
- [ ] 提供系统设置跳转链接
- [ ] 权限变更时实时更新状态

**预计周期**: 1 Cycle

---

### Phase 2: 技术债务清理 (P1) - 2 Cycles

#### TASK-215: Sidecar 健康监控
**当前问题**: Sidecar 进程崩溃后应用无响应

**解决方案**:
- 添加 sidecar 进程健康检查
- 崩溃后自动重启（指数退避）
- 重启失败时显示错误提示
- 最多重试 3 次

**验收标准**:
- [ ] Sidecar 崩溃后自动恢复
- [ ] 重启失败显示用户友好提示
- [ ] 最多重试 3 次
- [ ] 用户可手动重启

**Note**: 这解决了 TASK-163 阻塞问题

**预计周期**: 1 Cycle

---

#### TASK-216: WebSocket 重连优化
**当前问题**: WebSocket 断开后用户无感知

**解决方案**:
- 添加连接状态指示器（BackendStatusIndicator 已有）
- 断开时显示"重新连接中..."
- 重连期间消息队列
- 重连成功后自动发送队列消息

**验收标准**:
- [ ] 断开时显示"重新连接中..."
- [ ] 重连成功后自动恢复
- [ ] 用户可手动重试
- [ ] 队列消息不丢失

**预计周期**: 1 Cycle

---

### Phase 3: 多模型历史对比 (P2) - 延后到 v3.60.0

> **Critic 警告**: API 成本翻倍是硬伤，建议延后
> **替代方案**: 先实现"历史对比"（切换模型后重新生成，记录历史）

**预留任务**（延后）:
- TASK-217: 多模型历史对比 UI
- TASK-218: 对比结果保存
- TASK-219: 一键复制最佳

## 隐私检查

| 功能 | 涉及数据 | 处理方式 |
|------|----------|----------|
| 快捷键配置 | 按键组合 | localStorage |

## 架构概述

### 全局快捷唤起流程

```
用户按下 Ctrl+Shift+Space
         |
         v
  Tauri global-shortcut
         |
         v
  检查权限状态 (macOS)
         |
    +----+----+
    |         |
  有权限    无权限
    |         |
    v         v
 显示面板   显示引导
```

### 快速提问面板组件

```
QuickPanel (浮动窗口)
├── Header (模型选择器)
├── InputArea (输入框 + 发送按钮)
├── QuickActions (翻译/摘要/润色...)
└── ResponseArea (回复显示)
```
