```
## What makes it privacy-focused:

- **Zero telemetry** - No analytics, no tracking, no phone-home

- **Open source** - MIT license, audit the code yourself

- React 19 + TypeScript
- FastAPI Python backend
- 94% test coverage

Happy to answer any questions about the privacy architecture!
```

### r/selfhosted (1M users)

**Title**:
```
HuluChat - Self-hostable AI chat client, works with Ollama/LocalAI
```

**Body**:
```
Hi r/selfhosted,

HuluChat is a desktop AI chat client designed for self-hosters:

## Self-hosting features:

- **Works with local models**: Ollama, LocalAI, LM Studio, vLLM
- **No cloud dependency**: Just point to your local API
- **Local database**: SQLite, your data never leaves your machine
- **Zero telemetry**: No analytics, no tracking

## Quick Ollama setup:

1. `ollama serve`
2. In HuluChat settings:
   - Base URL: `http://localhost:11434/v1`
   - Model: `llama3` (or your preferred model)

Also supports any OpenAI-compatible API (DeepSeek, Together, etc.)

## Tech:

- Tauri 2.0 (Rust) - lightweight, ~50MB install
- React 19 + TypeScript
- FastAPI Python sidecar
- MIT license

GitHub: https://github.com/MrHulu/HuluChat

Would love feedback from the self-hosted community!
```

**Body**:
```
Hi r/LocalLLaMA,

I made HuluChat as a lightweight desktop client for running local models.

## Local model support:

- **Ollama**: `http://localhost:11434/v1`
- **LM Studio**: `http://localhost:1234/v1`
- **vLLM**: `http://localhost:8000/v1`
- **LocalAI**: Any OpenAI-compatible endpoint

Just set your local API URL and model name, and you're good to go.

## Why desktop over web UI?

- Native performance (no browser overhead)
- System-wide shortcuts
- Local file integration (RAG with local documents)
- True offline capability

## Other features:

- Session folders
- Global search
- Markdown/JSON export
- Dark mode
- 18 languages
- Cross-platform (Win/Mac/Linux)

Built with Tauri 2.0 (Rust) - ~50MB installed, no Electron.

GitHub: https://github.com/MrHulu/HuluChat

Let me know if you have questions about local model integration!
```
